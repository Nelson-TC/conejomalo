# ADR-003: Proveedor de Autenticación (Supabase/Auth0)

## Contexto
Se requiere un sistema de autenticación rápido de implementar, seguro y con mínimo mantenimiento.

## Decisión
Usar **Supabase Auth** o **Auth0** según disponibilidad del equipo, priorizando velocidad de integración.

## Consecuencias
- Reducción de tiempo de desarrollo.
- Dependencia de proveedor externo.
