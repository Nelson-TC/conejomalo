# ADR-007: Stack de testing (Vitest + Jest/Supertest)

## Contexto
Se requiere pruebas rápidas para componentes y endpoints críticos.

## Decisión
- **Frontend:** Vitest + Testing Library
- **Backend:** Jest + Supertest

## Consecuencias
- Cobertura mínima en flujo principal.
- Base escalable para QA futuro.
