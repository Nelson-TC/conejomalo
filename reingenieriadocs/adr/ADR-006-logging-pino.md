# ADR-006: Logging con Pino/Winston

## Contexto
Se necesita trazabilidad mínima de errores y eventos.

## Decisión
Usar **Pino** (preferente por performance) o **Winston** si se requiere mayor configuración.

## Consecuencias
- Observabilidad básica desde el inicio.
- Posible ajuste futuro según necesidades.
